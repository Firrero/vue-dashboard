This is a vue-based dashboard with AdminLTE integration
## Need to have
* Nodejs 
* npm

## Installing
from server folder:
```
npm install
```
from client folder:
```
npm install
```
## Running
from server folder:
```
npm start
```
from client folder:
```
npm run dev
```
### Using

Backend locates at:
```
http://localhost:8081
```
Frontend locates at:
```
http://localhost:8080
```
